### Which HTML Version ? 

There are HTML 4, XHTML 2.0, HTML 5.0, how does the browser know which HTML version to render? 

a directive `<!doctype> `(captial-case insensitive)

```
<!DOCTYPE HTML>
<!doctype html>
```



### Head & Body 

看不见的元素放到 `<head>` ，看得见的元素放到 `<body> `

```
<html> 
	<head>
		<title></title> 
	</head>
	<body> 
		... 
	</body> 
</html> 
```
